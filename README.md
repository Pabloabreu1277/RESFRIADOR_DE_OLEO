# RESFRIADOR_DE_OLEO
Esse projeto apresenta uma logica em ladder de um equipamento chamado, resfriador de óleo, com a logica de controle para bomba de óleo, compressor, ventilador e linha de segurança do sistema e equipamentos.

CLP = Micrologix 1100 serie B - 1763-L16BBB
Software = RSLogix 500

Lista de pontos:

Inputs
I/0 - AUTOMATICO
I/1 - SENSOR DE TEMPERATURA CONTATO
I/2 - PRESSOSTATO DE BAIXA PRESSÃO
I/3 - PRESSOSTATO DE ALTA PRESSÃO
I/4 - BOTÃO DE RESET DE ALARMES
I/5 - BOTÃO EMERGENCIA

Outputs
O/0 - BOTÃO LIGA BOMBA
O/1 - LIGA COMPRESSOR
O/2 - LIGA VENTILADOR
O/3 - LED ALARME
